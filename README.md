# Natural-Language-Processing

CS288 Natural Language Processing — UC Berkeley, Spring 2026.

Three assignments building up from classical text classification to a hybrid retrieval-augmented generation system.

| Folder | Assignment | Topic |
|--------|------------|-------|
| `Text_Classification/` | Assignment 1 | N-gram language modeling + Perceptron / MLP text classification on 20 Newsgroups and SST-2 |
| `Language_Modeling_and_QA/` | Assignment 2 | BPE tokenizer, decoder-only Transformer language model, and SQuAD question answering |
| `Hybrid_RAG_System/` | Assignment 3 | RAG QA over the Berkeley EECS website — BM25 + FAISS recall, RRF fusion, cross-encoder reranking, OpenRouter LLM generation |

See each subfolder's README for setup, usage, and details.
